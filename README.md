# sleepctl

Set percentage of sleeping players required to pass the night for Bukkit/Spigot.

![](sleepctl.png)

## Usage

```
/sleepctl [percentage]
```

Example:

```
/sleepctl 25
```

Set percentage to _at least_ 25%.

## Features
- Count overworld players only
- Persistent config file
- Vanilla bed behaviour
